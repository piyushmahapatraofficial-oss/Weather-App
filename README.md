# Version Control System (Git) - Notes

## What is Version Control?
Version control is a system that helps track changes in files over time and allows multiple people to collaborate efficiently.

## What is Git?
Git is a distributed version control system used to manage and track changes in source code.

## Basic Git Commands

### Initialize a repository
git init

### Add files to staging
git add .

### Commit changes
git commit -m "message"

### Connect to remote repository
git remote add origin <repo-url>

### Push to GitHub
git push -u origin main

## Key Concepts
- Repository: Storage location of project files  
- Commit: Snapshot of changes  
- Branch: Separate version of code  
- Merge: Combine branches
